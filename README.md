# Calculator Web Application

## Overview

This project is a simple calculator developed using HTML, CSS and JavaScript. The calculator performs basic arithmetic operations through an interactive user interface. CSS styling is embedded directly within the HTML file.

## Features

* Addition (+)
* Subtraction (-)
* Multiplication (×)
* Division (÷)
* Clear (C) functionality
* Real-time display updates
* Simple and responsive interface

## Technologies Used

* HTML5
* CSS3 (Embedded in HTML)
* JavaScript

## Project Structure

```text
Calculator/
```

## How It Works

1. Users enter numbers using the calculator buttons.
2. Mathematical operators can be selected.
3. The entered expression is displayed on the calculator screen.
4. Pressing the equals (`=`) button evaluates the expression.
5. The result is displayed instantly.

## Running the Project

1. Download or clone the repository.
2. Open the project folder.
3. Open `calculator.html` in any modern web browser.

No installation or external dependencies are required.
